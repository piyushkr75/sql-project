# World Layoffs Data Cleaning & EDA using MySQL

## Project Overview
Performed data cleaning and exploratory data analysis on a global layoffs dataset using MySQL.

## Skills Demonstrated
- Data Cleaning
- CTEs
- Window Functions
- Joins
- Aggregations
- Ranking Functions
- Exploratory Data Analysis

## Data Cleaning Steps
1. Removed duplicate records
2. Standardized company and country names
3. Handled null values
4. Converted date column to DATE datatype

## EDA Performed
- Total layoffs by company
- Total layoffs by industry
- Year-wise layoffs
- Rolling layoffs trend
- Top companies by layoffs each year
